# Dashboard Starter

Static dashboard starter for:
- Real-time appliance readings
- Historical power chart
- Alert list (threshold/abnormal)

## Run

From this folder:

```bash
python3 -m http.server 5500
```

Open:
- `http://localhost:5500`

By default, dashboard expects backend at:
- `http://localhost:8080`

Change in `app.js` if needed.
